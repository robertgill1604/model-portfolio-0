## üéØ Customization Guide

### Change Student Name
Find and replace "Robert" with your name throughout the files:
- HTML: Title, navbar logo, headings
- CSS: Logo styling if needed
- JavaScript: Console messages

### Update Contact Information
In `index.html`, locate the contact section and update:
- Email address
- Phone number
- LinkedIn profile
- GitHub profile

### Modify Projects
Edit the projects grid section in `index.html`:
- Change project titles and descriptions
- Update technology tags
- Modify GitHub/Live links

### Customize Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;      /* Change primary color */
    --secondary-color: #8b5cf6;    /* Change secondary color */
    /* ... other variables ... */
}
```

### Update Skills
Modify skill categories and tags in the Skills section of `index.html`
